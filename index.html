Question 1: Multiple Linear Regression (12 pts)

Model:

\[\hat{y} = 3.2 + 1.4X_1 - 0.9X_2 + 0.6X_3\]

a) Predict \(\hat{y}\) when \(X_1 = 2\), \(X_2 = 1\), \(X_3 = 0\)

\[ \hat{y} = 3.2 + 1.4(2) - 0.9(1) + 0.6(0) = 3.2 + 2.8 - 0.9 + 0 = 5.1 \]

Answer: 5.1

b) Interpret the coefficient of \(X_2\)

The coefficient of \(X_2\) is −0.9.

Interpretation: For every one-unit increase in \(X_2\), the predicted \(y\) decreases by 0.9 units, holding \(X_1\) and \(X_3\) constant.

c) Residual \(e = -1.5\), \(\hat{y} = 10.3\). Find actual \(y\)

\[ e = y - \hat{y} \Rightarrow y = \hat{y} + e = 10.3 + (-1.5) = 8.8 \]

Answer: 8.8

Question 2: Logistic Regression (14 pts)

Model:

\[\text{logit}(P(Y=1)) = -1.8 + 0.7X_1 - 1.2X_2\]

a) Compute \(P(Y=1)\) when \(X_1 = 3\), \(X_2 = 1\)

\[ \text{logit} = -1.8 + 0.7(3) - 1.2(1) = -1.8 + 2.1 - 1.2 = -0.9 \] \[ P(Y=1) = \frac{1}{1 + e^{0.9}} \] \[ e^{0.9} \approx 2.4596 \quad \Rightarrow \quad P = \frac{1}{1 + 2.4596} \approx \frac{1}{3.4596} \approx 0.289 \]

Answer: 0.289

b) Odds of \(Y=1\)

\[ \text{Odds} = \frac{P}{1-P} = \frac{0.289}{1-0.289} = \frac{0.289}{0.711} \approx 0.406 \]

Answer: 0.406

c) Factor change in odds when \(X_1 \uparrow 1\)

Coefficient of \(X_1\) = 0.7 → odds multiplier = \(e^{0.7}\) \[ e^{0.7} \approx 2.0138 \]

Answer: 2.014

Question 7: CART – Gini Impurity (12 pts)

Parent: 60 Class 0, 40 Class 1 (Total = 100)

a) Parent Gini

\[ p = \frac{40}{100} = 0.4, \quad \text{Gini} = 2(0.4)(0.6) = 0.48 \]

Answer: 0.48

b) Weighted Gini after split

Left: 50 Class 0, 10 Class 1 → \(p_L = 10/60 \approx 0.1667\)
Right: 10 Class 0, 30 Class 1 → \(p_R = 30/40 = 0.75\)

\[ \text{Gini}_L = 2(0.1667)(0.8333) \approx 0.2778 \] \[ \text{Gini}_R = 2(0.75)(0.25) = 0.375 \] \[ \text{Weighted} = \frac{60}{100}(0.2778) + \frac{40}{100}(0.375) = 0.1667 + 0.15 = 0.3167 \]

Answer: 0.317

Question 9: Neural Network – Forward Pass (16 pts)

Inputs: \(x_1 = 2\), \(x_2 = -1\)
Weights: \(w_1 = 0.5\), \(w_2 = -1.2\)
Bias: \(b = 0.3\)
Activation: ReLU

a) Pre-activation \(z\)

\[ z = 0.5(2) + (-1.2)(-1) + 0.3 = 1.0 + 1.2 + 0.3 = 2.5 \]

Answer: 2.5

b) After ReLU

\[ \text{ReLU}(2.5) = \max(0, 2.5) = 2.5 \]

Answer: 2.5

c) Final probability \(P(Y=1)\)

Output neuron: \(w_o = 1.5\), \(b_o = -0.5\)

\[ z_o = 1.5(2.5) - 0.5 = 3.75 - 0.5 = 3.25 \] \[ P(Y=1) = \frac{1}{1 + e^{-3.25}} = \frac{e^{3.25}}{1 + e^{3.25}} \approx 0.963 \]

Answer: 0.963
